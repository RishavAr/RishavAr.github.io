### 📚 Overview
Designed an AI-powered semantic search assistant to improve documentation retrieval for Confluence users.

### ⚙️ Implementation
- Used **sentence-transformers (all-MiniLM-L6-v2)** for embeddings  
- Implemented **FAISS-based vector retrieval**  
- Deployed via **FastAPI + Docker** for real-time access  

### 📊 Impact
- Achieved **85%+ retrieval precision**
- Improved engineering team productivity and documentation discoverability
